# Verwendung

Diese ZIP enthält eine neue `data.json` für deine GitHub-Pages-Lernplattform.

## Einsetzen

1. Öffne dein GitHub-Repository.
2. Lade `data.json` in den Hauptordner hoch.
3. Wenn dort schon eine `data.json` liegt, ersetze sie.
4. Commit changes.
5. Warte, bis GitHub Pages neu veröffentlicht hat.

## Neues Material ergänzen

Suche das passende Thema und ergänze im Feld `links` zum Beispiel:

[
  {
    "typ": "Lernpfad",
    "titel": "Interaktiver Lernpfad",
    "url": "material/mathematik/klasse-5/koerper-netze-schraegbilder-und-rauminhalt/lernpfad.html"
  },
  {
    "typ": "PDF",
    "titel": "Arbeitsblatt",
    "url": "material/mathematik/klasse-5/koerper-netze-schraegbilder-und-rauminhalt/arbeitsblatt.pdf"
  }
]

Die Datei `ORDNERSTRUKTUR.txt` zeigt dir, welche Ordner für die Themen sinnvoll wären.
